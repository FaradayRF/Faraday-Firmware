Detailed Description

FIFO circular buffer implementation using CC430 RAM.

These functions create a FIFO circular buffer using the onboard CC430 RAM. The circular buffers defined below are written as generic as possible to allow varying element lengths, counts, etc...

§ fifo_empty()

unsigned char fifo_empty ( fifo_state_machine *  buffer_struct)

Check if FIFO is empty.

This function is a simple method to check if the FIFO contains a waiting item to be retrieved.

Parameters
buffer_structA pointer to the fifo_sram_state_machine FIFO structure element containing the configuration data of the SRAM FIFO to check for a waiting FIFO item to retrieve
Return values
Thefunction returns a status byte indicating if the CC430 RAM FIFO supplied contains a waiting item to retrieve. Returns 1 if empty, 0 if item(s) waiting in FIFO.

Get a new data element from the CC430 RAM FIFO buffer.

This function is used to retrieve a new item (element) from the CC430 RAM FIFO buffer. Supplying the fifo_state_machine structure variable of the respective FIFO is required.

Parameters
buffer_structA pointer to the fifo_sram_state_machine FIFO structure element containing the configuration data of the SRAM FIFO to retrieve the FIFO item from
bufferA pointer to the buffer byte array used by the CC430 RAM FIFO
char_itemA pointer to the data byte/element that will be placed into the CC430 RAM FIFO.
Return values
Thefunction returns a status byte indicating a successful get_fifo() operation. Returns 1 if success, 0 if fail (likley an overflow).
