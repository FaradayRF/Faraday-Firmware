§ Clear_ADC12_Memory_Control_Register()

void Clear_ADC12_Memory_Control_Register ( unsigned char  MemoryChannel)

Clear the ADC12CTL2 memory register passed to the function.

This function clears the contents of the memory registers for the ADC12 peripheral. Passing the correct number to the function clears the respective register for ADC 12.

Parameters
MemoryChannelThe ADC 12 memory channel to clear 0-15.
